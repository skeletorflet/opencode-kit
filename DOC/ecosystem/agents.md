## Using Community Agents

### Installation

1. Clone the repository
2. Copy agent files to your `.opencode/agents/` directory
3. Or merge the configurations into your `opencode.json`

```bash
# Example: Install opencode-agents
git clone https://github.com/darrenhinde/opencode-agents.git
cp opencode-agents/agents/* .opencode/agents/
```

## Creating Your Own Agent Pack

Share your agent configurations with the community:

1. Create a GitHub repository
2. Include agent markdown files or JSON configs
3. Document each agent's purpose and usage
4. Submit to [awesome-opencode](https://github.com/awesome-opencode/awesome-opencode)
