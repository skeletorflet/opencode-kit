# UI/UX Pro Max

## What I Do

- Provide 50+ distinct visual styles beyond generic SaaS templates
- Offer 21 curated color palettes mapped to emotions and industries
- Guide design decisions with constraint-based frameworks
- Ensure designs are memorable, not template-like
- Apply anti-cliché rules to avoid AI design patterns

## When to Use Me

- Creating distinctive UI designs
- Choosing visual styles and palettes
- Avoiding generic/template-like designs
- Making design system decisions
- Creating unique brand identities

## Key Principles

1. **No Purple**: Purple/indigo/violet is the #1 AI design cliché
2. **No Standard Split**: Avoid 50/50 hero layouts
3. **Extreme Geometry**: Go sharp (0-2px) or very rounded (16-32px), never middle
4. **Animation Required**: Static designs fail. Use GPU-accelerated transforms
5. **Emotion Mapping**: Colors and typography must match intended emotion

## Anti-Cliché Rules

- ❌ No mesh/aurora gradients
- ❌ No glassmorphism as default
- ❌ No bento grids for simple content
- ❌ No deep cyan/fintech blue
- ❌ No "Empower/Elevate/Seamless" copy
- ❌ No 6-8px border-radius default

## Validation

Ask: "Would this exist in a Tailwind UI template?" If yes, redesign.
